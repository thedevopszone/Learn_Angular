# Learn_Angular

```
sudo npm install -g @angular/cli@latest
ng new my-first-app --no-strict
# Routing => no
# Stylesheet => CSS
cd my-first-app
npm install --save bootstrap@3
ng serve  # --open
http://localhost:4200
```


vi angular.json
```
"styles": [
              "node_modules/bootstrap/dist/css/bootstrap.min.css", # Add this line
              "src/styles.css"
            ],
```
